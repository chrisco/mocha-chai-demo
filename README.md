# A Short Mocha/Chai Testing Demo

## How I Made This

`````
$ mkdir tdd
$ cd tdd
$ npm init -y
$ npm install -g mocha
$ npm install --save-dev chai
Make and edit dinosaur.js and test/dinosaurTest.js files
````

## Run Tests

````
 $ mocha
````

## Notes

Can put "x" in front of "describe" or "it" to NOT run those tests.
